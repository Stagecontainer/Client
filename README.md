# React + Vite
이 프로젝트는 Vite, React, JavascriptSWC를 사용하여 구축되었습니다

## 요구사항
- Node.js (v21.5.0)
- Yarn (v1.22.19)

## 로컬 실행
- yarn install
- yarn dev(http://localhost:5173/)
